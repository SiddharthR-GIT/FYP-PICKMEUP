A Ride Sharing Application

Traveling is a key aspect of our lives today. But getting from one place to another can be a little tricky and expensive. This web application that connects drivers and passengers who have the same origin and similar destination. With the help of this application, passengers can be connected to drivers who are traveling to the same or similar destination.
